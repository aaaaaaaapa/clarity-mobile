# Clarity (mobile)

Мобильное приложение (React Native + Expo) для фиксации заявок о нарушениях санитарного состояния городской среды.

## Запуск

1) Установите зависимости

```bash
npm install
```

2) Укажите адрес API (если нужен бэкенд)

`src/utils/config.ts` → `API_BASE_URL`

3) Запустите Expo

```bash
npm run start
```

## Основные экраны

- **Карта**: метки + простая кластеризация, фильтры, создание заявки (кнопка «+» / долгий тап по карте).
- **Список**: список заявок, поиск и фильтры по статусу/категории.
- **Детали заявки**: просмотр и редактирование, кнопка «Показать на карте».
- **Статистика**: количество заявок (в том числе по статусам/категориям).
- **Профиль**: просмотр/редактирование данных пользователя.

## Фото (камера/галерея)

Для создания/редактирования заявки фото выбирается **только** через:

- «Сфотографировать» (камера)
- «Из галереи» (фотоплёнка)

Текущее API принимает `photo_link` как строку, поэтому выбранное изображение сохраняется как **data URI** (`data:image/...;base64,...`).

> Примечание: если в бэкенде у сущности Pin нет полей `category`/`status`,
> они сохраняются локально (AsyncStorage) по `id` метки.
